> Replace mode for [Kakoune].

## Installation

### [Pathogen]

``` kak
pathogen-infect /home/user/repositories/github.com/alexherbo2/replace.kak
```

## Usage

``` kak
map global user r -docstring 'Replace' ':<space>replace<ret>'
```

## Commands

- `replace`: Enter in replace mode
